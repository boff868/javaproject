# Java Project

A collection of Java data-structure and algorithm lab projects. The repository contains multiple small Maven-based exercises from the F28SG / OUC coursework sequence, including stacks, queues, linked lists, recursion, binary search trees, tries, priority queues, and sorting algorithms.

## Contents

Examples of included labs and demos:

```text
f28sg-ouc-lab1                         # Stack, calculator, and reverse exercises
f28sg-ouc-lab2                         # Recursion and tail-recursion exercises
f28sg-ouc-lab3                         # Linked list and stack practice
f28sg-ouc-lab4                         # Queue and doubly linked list practice
f28sg-ouc-demo-08-doubly-linked-lists  # Doubly linked list demo
f28sg-ouc-demo-10-binary-search-tree   # Binary search tree / map ADT
f28sg-ouc-demo-11-priority-queue       # Priority queue implementation
f28sg-ouc-demo-12-tries                # Trie implementation
f28sg-ouc-demo-13-insertion-sort       # Insertion, bubble, and quick sort practice
f28sg-ouc-demo-14-merge-sort           # Merge sort practice
```

## Requirements

- JDK 8 or newer
- Maven, for projects that include `pom.xml`

## How to Run Tests

Enter one lab/demo folder and run Maven tests:

```bash
cd f28sg-ouc-lab1
mvn test
```

Repeat this inside the specific lab folder you want to check.

## Notes

This repository is organized as a coursework archive rather than one single application. Each subfolder should be treated as an independent exercise with its own source and test files.
